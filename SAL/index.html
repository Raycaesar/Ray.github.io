How to Write Formulas

  • Use elements from Prop for atomic propositions.
  • Use ~ for negation. E.g., ~p represents "not p".
  • Use & for conjunction (AND). E.g., p & q represents "p and q".
  • Use + for disjunction (OR). E.g., p + q represents "p or q".
  • Use > for implication. E.g., p > q represents "p implies q".
  • Group formulas using parentheses. E.g., (p & ~q) + r represents "(p and not q) or r".
